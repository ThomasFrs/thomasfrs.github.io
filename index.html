/Games

/ThroneRoom

ThroneRoom is a simple medieval game about resource management.
I developed it when I started learning Python in April 2020 and have continuously seeked for improvement.

Release Date: 14/02/2022
Lastest Version: 1.1



/Heart Attack

Heart Attack is a top-down 2D shooter where you protect your heart from the forces of evil.
I developed this game in March 2021 as my first PC game using Unity, I later published it on Itch.io.

Release Date: 20/07/2021
Lastest Version: 1.0



/Retrowave

Retrowave is a minimalist driving game in set in a retro environment.
I developed this game in June 2020 as my first Mobile game using Unity, I then published it on the Play Store.

Release Date: 08/2020
Lastest Version: 1.2

/Steam Calculator

It is a small yet very useful program with a simple GUI made for calculations in the Steam Market.
I developed this program in October 2021 to help me when buying or selling items on the Steam Market.

Release Date: 30/10/2020
Lastest Version: 1.0
