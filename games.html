Games

I have been making games since I got in touch with Scratch.
Most can be considered unfinished but in the last couple of years,
I've had the chance to find out about Python and later on, C#.
These two languages have helped me create games I never thought
I could make on my own.

Retrowave

Language: C#

Retrowave is a minimalist driving game set in a retro environment.
I developed this game in June 2020 as my first Mobile game using Unity,
I then published it on the Play Store and it has over 1000 downloads.

Latest release: 08/2020
Release version: 1.2.0

Heart Attack

Language: C#

Heart Attack is a top-down 2D shooter where you must
protect your heart from the forces of evil.
I developed this game in March 2021 as my first PC game using Unity,
I later published it on Itch.io.

Latest release: 20/07/2021
Release version: 1.0.0

ThroneRoom

Language: Python

ThroneRoom is a simple medieval game about resource management.
I developed it when I started learning Python in April 2020
and I have continuously seeked to improve it.

Latest release: 21/02/2022
Current version: 1.1.0
